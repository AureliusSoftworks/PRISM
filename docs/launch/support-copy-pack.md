# Prism Support Copy Pack

Status: draft. Do not paste any `[PATREON_URL]` placeholder into public release
copy without replacing it with the live Patreon page URL.

This pack keeps Prism's support ask consistent: free official downloads first,
optional patronage second, no product access pressure.

## Support Position

Prism is free to download and use. If Prism becomes part of someone's real
workflow, they can optionally support continued development for `$5/month` on
Patreon.

Support does not unlock Prism, change LOCAL/ONLINE behavior, create priority
runtime access, add product tiers, or link a Patreon account inside the app.
It is simply a way to help fund continued development, release polish, and
public updates.

## Known Limitations Block

Known limitations for this launch:

- Installer polish and first-run setup are still being tightened.
- Some advanced modes remain exploratory or disabled until they are ready.
- Linux signing starts conservative; platform-specific warnings may still need
  clearer docs.
- Online model behavior depends on the provider keys and mode choices users
  configure themselves.
