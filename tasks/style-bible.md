# LocalAI Style Bible

## Bots Panel

- Treat the Bots panel as two animated views: the customizer for creating/editing one bot, and the Bot library for browsing many bots.
- The Bot library drawer opens from the customizer and keeps category filters persistent at the top.
- PRISM category tiles filter the bot list below; they do not navigate away from the category grid.
- Include a neutral theme-dependent All Bots tile alongside the five PRISM color families.
- Bot cards selected from the library should return the user to the customizer with that bot loaded for editing.
