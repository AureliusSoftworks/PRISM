# Feature 3

## AI Summary
<!-- kb:summary:start -->
This feature note is important in PRISM because it ensures that the system's internal models, such as title generation and summarization, remain consistent and secure by staying on a specific version of the Ollama model, even when using external services like OpenAI for user-facing chat. This helps maintain the integrity and trustworthiness of PRISM's internal operations.

## Feature description
**Dedicated system models** — user-facing chat can use local or OpenAI, but Prism's internal titles, starters, summaries, memory critic, and embeddings always stay local on mandatory Ollama models (`llama3.2` + `nomic-embed-text`).
