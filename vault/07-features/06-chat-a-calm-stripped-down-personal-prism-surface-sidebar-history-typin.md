## AI Summary
<!-- kb:summary:start -->
This feature matters in PRISM because it allows users to have a private and secure chat experience, with the option to switch between sending data locally or through the regular memory pipeline, depending on their online status. This provides an additional layer of control over user data and privacy.

## Feature description
**Chat** — a calm, stripped-down "personal Prism" surface (sidebar + history + typing + send). The only compose-adjacent control is an **Incognito** pill that doubles as an online/offline toggle: on = this send is local-only and bypasses memory; off = saved provider + normal memory pipeline.
