## Trigger

## Lesson
After changing API/backend behavior, especially memory extraction, inference, chat, or server routes, proactively restart the API dev server before asking for browser retests. If port 18787 is held by a stale Node process, clear that process and start `npm run dev -w apps/api` cleanly.
