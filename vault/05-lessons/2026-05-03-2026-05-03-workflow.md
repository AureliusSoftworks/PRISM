## Trigger
Memory directory flicker debugging regressed intentional visual styling through broad CSS suppressions instead of isolating the underlying breakpoint/layout cause.

## Lesson
When visual polish is already intentional, do not flatten/remove styling as a diagnostic fix unless explicitly approved. First preserve the visual baseline, reproduce the issue, and isolate layout/re-render/compositing causes with the smallest reversible probe.

## Applies to
LocalAI responsive UI/debugging work in `apps/web/src/app/page.tsx` and `apps/web/src/app/page.module.css`.
