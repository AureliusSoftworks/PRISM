## Trigger
Message actions were initially split into Resend, Revert, and Edit, but the desired interaction is a single Edit flow that rewinds from the edited message and sends the revised text.

## Lesson
For chat message correction, prefer one Edit action over separate Resend/Revert controls. Saving an edited user message should truncate the thread from that message, delete linked direct/compiled memories, and send the revised message for a fresh assistant reply.

## Applies to
`apps/web/src/app/page.tsx` message actions/composer edit mode and `apps/api/src/conversations.ts` memory cascade behavior.
